# 📱 Ticketing CRUD Android App

## 📌 Description

This is an Android application for managing tickets with CRUD functionality and PDF upload feature using Firebase.

## 🚀 Features

* Create, Read, Update, Delete (CRUD) Ticket
* Upload PDF to Firebase Storage
* Store data in Firebase Realtime Database
* Simple and clean UI

## 🛠 Tech Stack

* Java (Android)
* Firebase Realtime Database
* Firebase Storage
* Android Studio

## ⚙️ Setup Instructions

1. Clone this repository
2. Open in Android Studio
3. Add your `google-services.json`
4. Sync Gradle
5. Run the app

## ⚠️ Note

This project uses Firebase. You need to configure your own Firebase project.
